Question : Discuss the scope of var, let, and const

Answer : Var is old way of declaring variable in JS. Var is function scoped. While let is a newer way of delcaring variables in JS. Let is block scoped. const is Also newer addition to JS. It is also block scoped like let. While var and let's values can be changed or reassigned but const is constant can not be changed once initially delared.



Question : Tell us the use cases of null and undefined

Answer : When a variable returns null or undefined, null means the variable is empty or holds no value and undefined means the variable is declared but not initialized.



Question : What do you mean by REST API?

Answer : In REST API, REST stands for Representational State Transfer. Its a principle or a way to manage and send resources typically as JSON format. It sends a full representation of the resource as a agreed upon format over the network when that resource's unique identifier is used. Every resouerce has its own unique identifier (URI) It makes communication between server/backend and client/frontend easier.
